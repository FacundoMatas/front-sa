La construcción del modelo, asi como las tareas previas de recolección y porcesamiento de datos fueron realizadas utilizando la herramienta Jupyter Notebook, propiedad de Project Jupyter,y ofrece un entorno interactivo basado en tecnología web para crear documentos en los que se puede incluir código HTML, Python, Markdown, etc.
El código del modelo se encuentra en el repositorio publico de GitHub. Link al repositorio.

A fin de probar el modelo, se implementó una API Rest desarrollada con la librería Flask de Python, la cual tiene soporte para un método POST en el cual se envía el tweet que se desea clasificar, y devuelve una respuesta con el resultado de la clasificación.
El código del modelo se encuentra en el repositorio publico de GitHub. Link al repositorio.

Para facilitar el consumo de los servicios expuestos por el backend, se desarrolló una aplicación web en Angular que ofrece una interfaz para facilitar el ingreso del texto del tweet y el envío de la petición http correspondiente.
El código del modelo se encuentra en el repositorio publico de GitHub. Link al repositorio.
